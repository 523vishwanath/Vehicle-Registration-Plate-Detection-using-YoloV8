## 🗂️ Dataset

Download the **Vehicle Registration Plate** dataset from [Open Images](https://storage.googleapis.com/openimages/web/index.html) and unzip it. The raw layout is:

```
Dataset/
├── train/
│   └── Vehicle registration plate/
│       ├── *.jpg                 (5 308 images)
│       └── Label/
│           └── *.txt
└── validation/
    └── Vehicle registration plate/
        ├── *.jpg                 (386 images)
        └── Label/
            └── *.txt
```

Each `.txt` label file contains one or more rows in **Open Images format**:

```
Vehicle registration plate  xmin  ymin  xmax  ymax
```

The `prepare_dataset.py` script converts this to YOLO format and organises the data into the `vehicleDataset/` tree expected by the training config.

## ⚡ Quick Start

### 1 — Clone & install

```bash
git clone https://github.com/<your-username>/vehicle-registration-plate-detection.git
cd vehicle-registration-plate-detection
pip install -r requirements.txt
```

### 2 — Prepare the dataset

```bash
# Place Dataset.zip in the project root, then:
unzip Dataset.zip -d .

python scripts/prepare_dataset.py --dataset_root Dataset
# Creates vehicleDataset/ with YOLO-format images + labels
```

### 3 — Visualise ground-truth boxes

```bash
python scripts/visualize_ground_truth.py \
    --images_dir vehicleDataset/images/val \
    --labels_dir vehicleDataset/labels/val \
    --num_samples 4 \
    --output assets/ground_truth_preview.jpg
```

### 4 — Train

```bash
python scripts/train.py \
    --epochs 30 \
    --batch  8   \
    --imgsz  640

# Best weights → runs/detect/yolov8m/weights/best.pt
```

Or launch TensorBoard while training:

```bash
tensorboard --logdir runs/detect/yolov8m --reload_interval 5
```

### 5 — Inference on images

```bash
python scripts/inference.py \
    --model  runs/detect/yolov8m/weights/best.pt \
    --source vehicleDataset/images/val \
    --conf   0.25 \
    --show_results
```

### 6 — Inference on video

```bash
python scripts/inference.py \
    --model  runs/detect/yolov8m/weights/best.pt \
    --source path/to/your_video.mp4 \
    --conf   0.15 \
    --name   yolov8m_predict_video
# Output saved to runs/detect/yolov8m_predict_video/
```

### 7 — COCO evaluation

```bash
python scripts/evaluate.py \
    --model runs/detect/yolov8m/weights/best.pt \
    --data  vehicleRegistration.yaml \
    --plots
```
