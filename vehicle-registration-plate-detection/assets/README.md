# Assets

Place your inference demo GIF or video here.

Recommended file:
- `assets/demo.gif` — a short animated GIF of inference running on a video clip
- `assets/ground_truth_preview.jpg` — generated by `scripts/visualize_ground_truth.py`

## How to create a demo GIF from the inference output video

```bash
# Install ffmpeg if needed
sudo apt install ffmpeg

# Convert mp4 → gif (first 10 seconds, 15 fps, 640 wide)
ffmpeg -i runs/detect/yolov8m_predict_video/output.mp4 \
       -t 10 -vf "fps=15,scale=640:-1:flags=lanczos" \
       -loop 0 assets/demo.gif
```

Then commit it:
```bash
git add assets/demo.gif
git commit -m "Add inference demo GIF"
```
